# Vercel Deployment Steps

## Step 1: Prepare GitHub Repository

Create a new GitHub repository and upload all files from this project folder.

## Step 2: Import Project in Vercel

1. Open Vercel.
2. Click **Add New Project**.
3. Choose the GitHub repository.
4. Vercel should detect it as a Next.js project.
5. Click **Deploy**.

## Step 3: Review Deployment

After deployment, Vercel will provide a website URL similar to:

```text
your-project-name.vercel.app
```

Open the link and check:

- Homepage
- Monitoring Tools pages
- Parent-Child Camp pages
- AI Voice Questions page
- Templates page
- Schedule page
- FAQ page

## Step 4: Add Protection if Internal Only

If the website is internal use only, consider enabling available Vercel protection options or placing the site behind an internal access process.

## Step 5: Add Custom Domain Later

If needed, add a custom domain under:

```text
Vercel Project Settings > Domains
```
