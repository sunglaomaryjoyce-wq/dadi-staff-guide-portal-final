# Recommended Future Features

This first version is a guide website. Later, you may upgrade it into a real monitoring system.

- Password protection

- Separate admin pages

## Phase 3: Online Monitoring Records

- Staff submit checking records online
- OIC review status
- Monthly monitoring summary
- Department compliance dashboard

- users
- departments
- employees
- monitoring_records

- camp_checklists
- uploaded_files

- Monthly JR compliance report

- Calendar random checking report
- Work Plan OGSA review report
- KPI formatting error report
- Parent-Child Camp readiness checklist
